# Money Tracker

Веб-приложение для учёта личных расходов.

## Возможности

- Авторизация через Google OAuth 2.0
- Добавление расходов
- Удаление расходов
- Просмотр истории расходов
- Пагинация списка расходов

## Технологии

- React
- TypeScript
- Node.js
- Express
- MongoDB
- Google OAuth 2.0
- JWT
- Vercel
- Render
